Alegerea_masinii
Un program simplu în C# care demonstrează utilizarea instrucțiunii switch case pentru selectarea unui model de mașină dintr-o listă.

Ce face programul:
Afișează o listă de mărci auto disponibile.

Cere utilizatorului să introducă o opțiune.

Folosește switch pentru a afișa un mesaj specific sau prețul pentru mașina aleasă.

Include un caz default pentru situația în care utilizatorul introduce o mașină care nu este în listă.

Exemple de cod folosite:
Console.ReadLine() pentru citirea alegerii.

Structura switch(variabila) pentru logică.

case "Marca": pentru fiecare opțiune în parte.

Cum se rulează:
Descarcă fișierul Program.cs.

Rulează-l folosind comanda dotnet run.
